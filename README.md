# This Is How They Know

![This Is How They Know Banner](./public/assets/images/banner.png)

**A comprehensive media analysis and fact-checking platform promoting transparency in journalism and empowering readers with critical thinking tools.**

In an era of rapid-fire misinformation, **This Is How They Know (TIHTK)** provides the essential infrastructure for truth. Built with a high-performance NestJS backend and a sleek, modern Next.js/React frontend, TIHTK bridge the gap between complex data analysis and accessible journalism.
